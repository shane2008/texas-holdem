To launch the program:

1. Install ruby 2.2 or higher. Once installed 'ruby -v' should return ruby version.
This exercise is using ruby 2.2.2p95.

2. Install Ruby testing package 'Rspec' using 'gem install rspec'.

3. In project root, type 'rspec spec' to run all tests. All 28 tests/specs should pass.

4. In shell type './texas_holdem_runner.rb' or 'ruby texas_holdem_runner.rb' to launch the program
